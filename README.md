# ⚡ DoS Attack Simulator

> Educational Linux-based cybersecurity project for demonstrating multi-vector Denial-of-Service attack simulations in controlled lab environments.

---

## 📌 About

This project is a Bash-based cybersecurity lab tool designed to demonstrate how network traffic flooding and packet-based stress-testing techniques work internally.

The script provides an interactive Linux terminal experience with multiple traffic simulation vectors using common networking utilities.

---

## 🚀 Features

- Multi-vector traffic simulation
- Colored terminal interface
- Root privilege verification
- Linux networking tool integration
- Packet crafting demonstrations
- Educational cybersecurity workflow
- Lightweight Bash automation

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/roshandhiman/dos_attack.git
cd DoS-Attack-Simulator
```

Give execute permission:

```bash
chmod +x attack.sh
```

Install dependencies (Fedora):

```bash
sudo dnf install hping3 curl netcat iptables -y
```

Run script:

```bash
sudo ./attack.sh
```
